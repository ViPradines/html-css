Código-fonte / Pre-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript, e o código-fonte fica entre a tag CODE.

Podemos utilizar a tag PRE (Pré-Formaração), que considera tudo que foi escrito no código e mostra na pagina HTML, inclusive as quebras de linha no código.

Exemplo de mescla entre as tags PRE e CODE:

        
        num = int(input('digite um número'))
        if num % 2 == 0:
            print(f'O número {num} é PAR.')
        else:
            printf(f'O número {num} é ÍMPAR.')
        print('Fim do programa')
